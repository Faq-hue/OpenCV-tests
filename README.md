# OpenCV Tests Repository

This repository contains basic tests using the OpenCV library for computer vision tasks.

## 📌 Description

The project includes simple experiments and implementations using OpenCV, focusing on:

- 😊 Smile detection
- ✋ Hand detection
- 🎨 Color detection

These tests are intended for learning and understanding fundamental computer vision concepts using OpenCV.

## 🛠️ Technologies

- Python
- OpenCV

## 🚀 Purpose

The goal of this repository is to practice and explore basic computer vision techniques such as face feature detection, hand tracking, and color recognition.
